# YCLIENTS Landing — папка с сайтом

## Файлы
- index.html — главная страница (открой двойным кликом, чтобы посмотреть локально)
- assets/ — папка с картинками (4 файла, ~211 КБ)

## Как опубликовать на нормальный домен (бесплатно, за 1 минуту)

### Вариант 1: Netlify Drop (самый простой)
1. Открой https://app.netlify.com/drop
2. Перетащи туда ВСЮ папку yclients-site-folder целиком
3. Получишь ссылку типа https://elegant-fox-a1b2c3.netlify.app
4. В настройках можно переименовать в alenushka-yclients.netlify.app

### Вариант 2: Vercel
1. Зарегистрируйся на vercel.com
2. New Project → Import → перетащи папку
3. Готово

### Вариант 3: Любой хостинг
Залить папку через FTP/SFTP в корневой каталог сайта.
